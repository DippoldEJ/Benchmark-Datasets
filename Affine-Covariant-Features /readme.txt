
# Affine Covariant Features Dataset
DippoldEJ Overview Benchmark Dataset with explanation, link and reference. <br />
<br />
This is a deeper look into the Affine Covariant Features Dataset. This includes the actual visual data, available metadata and an example application puplished by myself. This dataset was a vital part of me proofing, convincingly testing and scientifically verifing my findings. 

Data Availability
-----------------
The dataset consits of 8 study cases, each including 5 images with homograpy matrix of each pair (img1-img2, img1-img3, ... ). The 8 study cases covering different scene content and different manipulations. 

Dowload Link: https://www.robots.ox.ac.uk/~vgg/research/affine/#software/
